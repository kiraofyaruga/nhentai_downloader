# How To Use NHentai Downloader

## Step 1: Install Python

Go to python.org and download Python 3.7 or higher. During installation, check "Add Python to PATH".

## Step 2: Download the Script

Save nhentai_gui.py or nhentai.py to a folder on your computer.

## Step 3: Run the Script

Double-click the script file, or open Command Prompt in the folder and type:

python nhentai_gui.py

or

python nhentai.py

## Step 4: Enter Gallery ID

Type the gallery ID or URL and press Enter.

Examples:
- 123456
- 123456, 789012, 345678
- https://nhentai.net/g/123456/

## Step 5: Choose Output Format

Select ZIP, CBZ, or PDF.

## Step 6: Wait for Completion

The script will download all images and create your file in the downloads folder.

## Step 7: Find Your Files

Open the downloads folder next to the script. Your ZIP, CBZ, or PDF files are there.
